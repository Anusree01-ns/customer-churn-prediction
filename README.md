(https://customer-churn-predictor-01.streamlit.app/)

## 📌 Project Overview

This project predicts whether a telecom customer is likely to churn (leave the service) using Machine Learning.

The goal is to help businesses identify at-risk customers and improve retention strategies.

## 📈 Model Performance

| Model | Accuracy |
|---------|----------|
| Logistic Regression | 80.31% |
| Random Forest | 78.68% |
| XGBoost | 81.81% |

ROC-AUC Score: **0.836**

---

## 💡 Business Insights

- Customers with low tenure are more likely to churn.
- Higher monthly charges increase churn probability.
- Long-term contracts improve retention.
- Fiber optic customers show higher churn rates.
- Total charges and tenure are strong churn indicators.
